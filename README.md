Este proyecto fue desarrollado para practicar javascript puro sin librerias o frameworks:

Manejo del DOM,
Programación asincrónica (async/await),
Arquitectura modular en frontend,
Separación de responsabilidades (mapper, state, UI),
Consumo de APIs REST.

Problemas resueltos:

Manejo del backend(db.json) string vs number, snake_case vs camelCase,
Sincronización del estado con el backend,
Control de paginación cuando no hay datos,
Autenticación básica implementada,
Actualización de usuarios,
Validación de formularios en tiempo real gracias a la herramienta VITE

Mejoras futuras:

Optimización de rendimiento usando librerias o frameworks,
